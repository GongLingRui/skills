## 什么是 LingRui？

LingRui 是一个基于 **Agentic Context Engineering (ACE)** 框架和 **ReAct 编排模式**的智能多 Agent 协作系统。

简单来说，它就像你拥有了一个专业的**内容创作团队**——

- 有洞察分析师，帮你研究市场趋势和用户需求
- 有爆款策划师，帮你找到传播的关键点
- 有文案撰稿人，为你创作各种平台的内容
- 有质量把关人，确保输出内容的专业性

而且这个"团队"可以 24 小时待命，批量高效产出。

### 🤖 17 位专业 Agent 协同工作

LingRui 不是简单的 AI 写作工具，而是真正的多 Agent 协作系统：

**内容生成类**：XHSPostAgent、TikTokScriptAgent、WeChatArticleAgent、TwitterPostAgent、InstagramPostAgent

**分析洞察类**：InsightAgent、HitpointAgent、PersonaAgent、ContentAnalysisAgent、BrandAnalysisAgent、ResonantAgent

**工具支持类**：WebSearchAgent、KnowledgeAgent、RevisionAgent、EvaluationAgent、FakeThinkAgent

它们各司其职，协同配合，只为产出最优质的内容。

### 💾 智能记忆系统

基于最新的 AI Agent 研究成果，LingRui 拥有先进的记忆增强系统：

- **主动记忆管理**：自动学习你的偏好
- **会话连贯性**：记住之前的对话，保持一致
- **多维度评分**：智能判断信息重要性
- **遗忘曲线模拟**：像人脑一样优化记忆存储

### ⚡ 批量生产，效率倍增

- 支持高并发处理（最多 3 个任务同时进行）
